Skip Counting by 10 s



Let Us Learn

Listen to your teacher skip count by l0s. Repeat after him or her.


Let Us Try

A.
Draw l0 circles on each hat. Then skip count by l0s. Write the number on the line. The first one has been done for you.

B.
Skip count by l0s. Write each missing number on the line.
