Addition with Three Addends



Let Us Learn

Look at each picture. Read the sentence about it. Then answer the question that follows.

How many lollipops are there in all?

To get the answer, put together the addends. 3 + 5 + 4

Add the first addend 3 to the second addend 5. 3 + 5 = 8

Add the sum of the first two addends to the third addend 4.

You can see from the example that there are two ways to write addition sentences. You can write them in horizontal or in vertical form.
